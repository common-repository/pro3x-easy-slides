== Description ==

Simple and free slider plugin for Wordpress. Use this plugin to add featured content slider to your theme. You can create slides simply by draging Pro3x Easy Slides widgets to Pro3x Easy Slides Sidebar and you can create as many slides as you like. This plugin is wrapper for jQuery Easy Slider script developed by Alen Grakalic.

== Installation ==

1. Use Wordpress administration or manually upload plugin files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action("displayslider"); ?>` in your themes index.php template to display slider
1. Open plugin editor, select Pro3x Easy Slides from drop down list and customize easySlider.css
1. Go to you Widgets page and add Pro3x Easy Slides widget to Pro3x Easy Slides Sidebar to create slides
